Side Tabs

If you are using or want to use side tabs for radio input options after DOM has loaded, you can initialise them by calling SUI.sideTabs( this ); for each .sui-side-tabs label.sui-tab-item input element.

Side tabs are used more like a selector. They can used for yes/no, or more usually for choosing which option out of 2+ to choose from and then showing content for that choice.

Documentation

Default This is the markup required for standard tabs.

Tabs markup changed to follow WAI-ARIA best practices. We still support old legacy code but we strongly recommend to update your project as soon as possible to prevent any problems in the future.

<div class="sui-tabs">

	<div role="tablist" class="sui-tabs-menu">

		<!--
		NOTE(S):

		1. Include "type" attribute when using tabs inside a form to prevent
		problems when submitting the form.

		2. Include "aria-controls" attribute when using tabs with content.

		3. Unique IDs are a must for button tab and its content.
		-->

		<button
			type="button"
			role="tab"
			id="tab-unique-id"
			class="sui-tab-item active"
			aria-controls="tab-content-unique-id"
			aria-selected="true"
		>
			Active Tab
		</button>

		<button
			type="button"
			role="tab"
			id="tab-unique-id"
			class="sui-tab-item"
			aria-controls="tab-content-unique-id"
			aria-selected="false"
			tabindex="-1"
		>
			Inactive Tab
		</button>

	</div>

	<div class="sui-tabs-content">

		<div
			role="tabpanel"
			tabindex="0"
			id="tab-content-unique-id"
			class="sui-tab-content active"
			aria-labelledby="tab-unique-id"
		>
			<p>Content. Active Tab.</p>
		</div>

		<div
			role="tabpanel"
			tabindex="0"
			id="tab-content-unique-id"
			class="sui-tab-content"
			aria-labelledby="tab-unique-id"
			hidden
		>
			<p>Content. Inactive Tab.</p>
		</div>

	</div>

</div>

Advanced Usage 
SUI.tabs({
	callback: function( tab, panel ) {
		console.log( tab );
		console.log( panel );
	}
});

callback

It's an anonymous function that will run when you click a tab. You will have access to the clicked tab and the respective panel (tab content).

To get all tabs within the group you can do this inside the anonymous function:

var wrapper = tab.closest( '.sui-tabs' ),
	tablist = tab.closest( '[role="tablist"]' ),
	tabs    = tablist.find( '> .sui-tab-item' ),
	panels  = wrapper.find( '> .sui-tabs-content > [role="tabpanel"]' )
	;

	console.log( tabs );
	console.log( panels );
