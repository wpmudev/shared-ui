Preview

The form field element has been created to wrap, as a container, each form field. This will guarantee us that each field in a form has the correct alignment between its own elements and other fields elements.

Do

Place one component (and its elements) per each form field container.

Don't

Documentation

Specifications

Version ^2.0.0 This is the minimum required version of SUI to use with this element in order for it to work properly.
Utilities Unsupported This will tell you if the element supports the modifier classes listed on Utilities page.
Monochrome Not Apply This will tell you if the element supports the Colour Accessible option found on our plugins' Settings page.
RTL Language Not Apply This will tell you if the element supports right-to-left langues.
