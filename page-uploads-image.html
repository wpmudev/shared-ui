Markup 
<div class="sui-form-field">

	<label class="sui-label">Label</label>

	<div class="sui-upload">

		<input
			type="file"
			value="filename.png"
			readonly="readonly"
		/>

		<div class="sui-upload-image" aria-hidden="true">
			<div class="sui-image-mask"></div>
			<div role="button"
				class="sui-image-preview"
				style="background-image: url('path-to-image/filename.png');">
			</div>
		</div>

		<button class="sui-upload-button">
			<i class="sui-icon-upload-cloud" aria-hidden="true"></i> Upload file
		</button>

		<div class="sui-upload-file">

			<span>filename.png</span>

			<button aria-label="Remove file">
				<i class="sui-icon-close" aria-hidden="true"></i>
			</button>

		</div>

	</div>

	<!-- Repeat "sui-upload" content for multiple files -->

	<!-- Use "sui-notice" for error message. -->

</div>
Behavior

1. When clicking on .sui-upload-button element, we should open WordPress Media Library or browser files finder for user to select the desired file to be uploaded.

2. When clicking on .sui-upload-file > button we should remove / clear uploaded image.
