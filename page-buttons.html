Interactive Demo

Buttons are used everywhere across all designs to highlight actions. Generally, the rule is one Solid coloured button per screen, where the Ghost buttons can be used more freely throughout screens as a secondary actrion.

The demo below lets you preview the button component, its variations, and configuration options. Each tab displays a different type of button.

Text Button

These buttons have less visual prominence, so should be used for low emphasis actions.

<button class="sui-button">
	<span class="sui-button__label">Text Button</span>
</button>

<button class="sui-button sui-button--icon-leading">
	<span class="sui-button__icon sui-icon-wpmudev-logo sui-sm" aria-hidden="true"></span>
	<span class="sui-button__label">Text Button</span>
</button>

<button class="sui-button sui-button--icon-trailing">
	<span class="sui-button__label">Text Button</span>
	<span class="sui-button__icon sui-icon-wpmudev-logo sui-sm" aria-hidden="true"></span>
</button>

Solid Button

This is the most prominent button because of its contrasting surface color. It is also used as a primary action button.

<button class="sui-button sui-button--solid">
	<span class="sui-button__label">Solid Button</span>
</button>

<button class="sui-button sui-button--solid sui-button--icon-leading">
	<span class="sui-button__icon sui-icon-wpmudev-logo sui-sm" aria-hidden="true"></span>
	<span class="sui-button__label">Solid Button</span>
</button>

<button class="sui-button sui-button--solid sui-button--icon-trailing">
	<span class="sui-button__label">Solid Button</span>
	<span class="sui-button__icon sui-icon-wpmudev-logo sui-sm" aria-hidden="true"></span>
</button>
Solid Button Color Variations
Color Name CSS Class
Grey (Default) sui-button--solid
Blue sui-button--solid-blue

Purple sui-button--solid-purple
Green sui-button--solid-green
Note: Color classes listed here cannot be used together. That means, if you need a blue button, you must replace the default class with the blue class and vice-versa.

Color variations are the same used on Ghost Button. The only difference is that this button has always present the class: sui-button--dashed.

Disabled

To disable a button, add the disabled attribute directly to the button element.

<button class="sui-button sui-button--solid" disabled>
	<span class="sui-button__label">Disabled Button</span>
</button>

CSS Classes

CSS Class Description
sui-button Mandatory. Defaults to a text button.
sui-button--md Styles a button that has 50px height.
sui-button--lg Styles a button that has 60px height.
sui-button--xl Styles a button that has 70px height.
sui-button--full-width Styles a button that occupies the full width of its container.
sui-button__icon-leading Styles a button with a leading (left) icon.
sui-button__icon-trailing Styles a button with a trailing (right) icon.
sui-button__label Mandatory. Indicates the element containing the button's text label.
sui-button__icon Indicates the element containing the button's icon.

Compound Button

About In some parts of the design you gonna find buttons transformed into icons only buttons on small screens.
Regular Button This is the basic structure of a compound button, text with icon on desktop and icon only on small screens. 
<button type="button" class="sui-button sui-button-compound">

	<!-- Content for large screens -->
	<span class="sui-compound-desktop" aria-hidden="true">
	<span class="sui-icon-academy"></span>
	Compound Button
	</span>

	<!-- Content for small screens -->
	<span class="sui-compound-mobile" aria-hidden="true">
	<span class="sui-icon-academy"></span>
	</span>

	<!-- Button label for screen readers -->
	<span class="sui-screen-reader-text">Compound Button</span>

</button>
Loading Button This is a regular compound button that includes an icon spinning animation when an action is being performed, e.g. when saving changes. Button on default state: Button on loading state: 
<!--
NOTE:
To put this button on "loading" state just include
`sui-button-onload` class next to `sui-button`.
-->
<button class="sui-button sui-button-compound">

	<!-- Content Wrapper -->
	<span class="sui-loading-text" aria-hidden="true">

		<!-- Content for large screens -->
		<span class="sui-compound-desktop">
		<span class="sui-icon-academy"></span>
		Compound Button
		</span>

		<!-- Content for small screens -->
		<span class="sui-compound-mobile">
		<span class="sui-icon-academy"></span>
		</span>

	</span>

	<!-- Button label for screen readers -->
