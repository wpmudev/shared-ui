Sticky box

Stick the box to the top of the page when scrolling on large screens. For small screens it will just highlight adding a shadow to the box but won't stick.

Click on the button below to add more sample boxes at the end to increase the amount of content on the page, then scroll and try the sticky sidebar.

Base Structure This white-box contains several box-elements depending on what you need. 
<!-- Static Box -->
<div class="sui-box">
	...
</div>

<!-- Sticky Box -->
<div class="sui-box sui-box-sticky">
	...
</div>
Box Header This is a sui-box element. 
<div class="sui-box-header">

	<!-- Box title with icon -->
	<h3 class="sui-box-title">
		<i class="sui-icon-hummingbird" aria-hidden="true"></i> Box Title
	</h3>

	<!-- Elements aligned to left, next to title -->
	<div class="sui-actions-left">
		...
	</div>

	<!-- Elements aligned to right -->
	<div class="sui-actions-right">
		...
	</div>

</div>
