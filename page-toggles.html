States

Toggles are usually used for on/off functions where the use case is simply turning something on or off. Optionally you can show sub-settings when the toggle is turned on.

Off - Default

The default state for all inputs, and textareas.

Off - Disabled

The toggle off & disabled state.

On - Active

The toggle on / active state.

On - Disabled

The toggle is on but in disabed state.

Variations

Toggles can be used with and without description, or there might be cases when you just need the toggle without the label.

Label - Default

By default toggles must have a label.

Label - Hidden

The label is visually hidden but available for screen readers.

Description - Default

The toggle with label and description.

Settings

Some toggles may need to show additional settings when turned on.

Off - Default

The default state keeps the settings hidden.

On - Active

Once toggle is active settings are shown.

Documentation

Base Markup

This is the main content for toggle button.

<div class="sui-form-field">

	<label for="unique-id" class="sui-toggle">

		<!--
		ELEMENT: Toggle input.

		1. Include "aria-labelledby" when toggle has a label.
		2. Include "aria-describedby" when toggle has a description.
		-->
		<input
			type="checkbox"
			id="unique-id"
			aria-labelledby="unique-id-label"
			aria-describedby="unique-id-description"
		/>

		<!-- ELEMENT: Toggle slider. -->
		<span class="sui-toggle-slider" aria-hidden="true"></span>

		<!-- ELEMENT: Toggle label. -->
		<span id="unique-id-label" class="sui-toggle-label">Toggle Label</span>

		<!-- ELEMENT: Toggle description. -->
		<span id="unique-id-description" class="sui-description">Toggle description goes here.</span>

	</label>

</div>

Sub Settings Markup

This is the markup for toggles with sub-settings.

<div class="sui-form-field">

	<!--
	NOTE:
	Here should go "sui-toggle" but notice the new attribute you gonna need
	to add for better accessibility: aria-controls.
	-->
	<label class="sui-toggle">
		<input aria-controls="unique-id-content" />
	</label>

	<!--
	NOTE:
	Include "display: none" when toggle settings content is hidden and remove
	it when you need to show it.
	Don't forget to include "aria-label" for a quick / brief description of the
	content inside.
	-->
	<div
		tabindex="0"
		id="unique-id-content"
		class="sui-toggle-content"
		aria-label="A brief description"
		style="display: none;"
	>
		... Content ...
	</div>

</div>
