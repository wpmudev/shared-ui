Design

We have 2 different designs for radio buttons: text only and text with image. You can get text only design using sui-radio class but if you need to add an image is better to use sui-radio-image class.

Below you can find both designs. Click on the tabs to get a visual example and demo code.

Text only (Default)

Alignment

There are two ways to align radio buttons: inline and stacked. By default radio buttons are aligned one next to another as inline elements, but you can pile them one on top of another as stacked elements adding sui-radio-stacked class to sui-radio element.

Sizes

Radio options have 2 font size variations: default that has 15px font size, and small that has 13px font size. To have smaller font size you just need to add sui-radio-sm class to sui-radio element.
