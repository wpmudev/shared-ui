Container

Progress bar has two different visual containers: default, which is the main container for progress bar and all the other elements; and boxed, which adds a border around progress bar to visually highlight it.

Default
50% 
<div class="sui-progress">
	<!-- Place progress elements here -->
</div>

Elements

Progress bar consists of 5 elements: loading wheel, percentage text, loading bar, cancel button and status text. You can modify the element depending on the design you need to replicate.

Loading Wheel 
<span class="sui-progress-icon" aria-hidden="true">
	<span class="sui-icon-loader sui-loading"></span>
</span>
Percentage Text
50% 
<span class="sui-progress-text">50%</span>
Loading Bar 
<div class="sui-progress-bar" aria-hidden="true">
	<span style="width: 50%"></span>
</div>
Cancel Button 
<button
	class="sui-button-icon sui-tooltip"
	data-tooltip="Cancel"
>
	<span class="sui-icon-close" aria-hidden="true"></span>
</button>
