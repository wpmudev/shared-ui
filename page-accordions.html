Documentation

If you are using individual js components, accordion.js is required.

If you want to perform some actions, like open a menu or adding a link, inside accordion items header remember to add sui-accordion-item-action to the element, that way when user clicks on the item it won't open/close the accordion but instead perform the action you want to like openning a dropdown menu with more options.

Flexbox Accordions 
<div class="sui-accordion">

	<div class="sui-accordion-header">
		<div>Recommendation</div>
		<div>Score / 100</div>
		<div>Type</div>
	</div>

	<div class="sui-accordion-item">

		<div class="sui-accordion-item-header">
			<div class="sui-accordion-item-title"><span aria-hidden="true" class="sui-icon-warning-alert sui-error"></span> Item</div>
			<div><div class="sui-circle-score sui-grade-f" data-score="8"></div></div>
			<div>More info on something <button class="sui-button-icon sui-accordion-open-indicator" aria-label="Open item"><span class="sui-icon-chevron-down" aria-hidden="true"></span></button></div>
		</div>

		<div class="sui-accordion-item-body">
			<div class="sui-box">
				<div class="sui-box-body">
					<p>Item content – Hello!</p>
				</div>
			</div>
		</div>

	</div>

</div>
Table Accordions 
<table class="sui-table sui-accordion">

	<thead>
		<tr>
			<th>Recommendation</th>
			<th>Score / 100</th>
			<th>Type</th>
		</tr>
	</thead>

	<tbody>

		<tr class="sui-accordion-item">
			<td class="sui-table-item-title"><span aria-hidden="true" class="sui-icon-warning-alert sui-error"></span> Item</td>
			<td><div class="sui-circle-score sui-grade-f" data-score="6"></div></td>
			<td>More info on something. <button class="sui-button-icon sui-accordion-open-indicator" aria-label="Expand"><span class="sui-icon-chevron-down" aria-hidden="true"></span></button></td>
		</tr>

		<tr class="sui-accordion-item-content">
			<td colspan="3">
				<div class="sui-box">
					<div class="sui-box-body">
						<p>Item content – Hello!</p>
					</div>
				</div>
			</td>
		</tr>

	</tbody>

</table>

Disabled Item Disable the row and prevent from opening.

Disable a flexbox accordion item:

<div class="sui-accordion-item"> ... </div>
<div class="sui-accordion-item"> ... </div>
<div class="sui-accordion-item sui-accordion-item--disabled"> ... </div>

Disable a table accordion item:

<tbody>
	<tr class="sui-accordion-item sui-accordion-item--disabled"> ... </tr>
	<tr class="sui-accordion-item-content"> ... </tr>
</tbody>
