Design

We have 2 different designs for checkboxes: text only and text with image. You can get text only design using sui-checkbox class but if you need to add an image is better to use sui-checkbox-image class.

Below you can find both designs. Click on the tabs to get a visual example and demo code.

Alignment

Checkboxes can be aligned in two different ways: inline and stacked. By default these are aligned one next to another as inline elements, but you can also pile them one on top of another as stacked elements adding sui-checkbox-stacked class to sui-checkbox element.

Inline (Default)
Stacked

<label class="sui-checkbox sui-checkbox-stacked"></label>

Sizes

Checkboxes have two font size variations: default which has 15px font size, and small which has 13px font size. To have smaller font size you just need to add sui-checkbox-sm class to sui-checkbox element.

Font 15px (Default)
Font 13px (Small)
