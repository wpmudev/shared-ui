Requirements
Remember you can have a maximum of 3 banners in the cross sell section. To change them you will do it using your SCSS file: 
/**
* Cross Sell Banners
* In lowercase insert the name of the plugin you want to promote.
*
* NOTE: This code must go on your SCSS file before you import shared-ui.
*/

$cross-sell-1: 'smush';
$cross-sell-2: 'defender';
$cross-sell-3: 'smartcrawl';
