Special

There are times when you need a top-center tooltip displayed on desktops, but on small screens content is being cut-off.
You can add sui-tooltip-mobile to keep original position but constrain content on small screens only.
You can add sui-tooltip-{position}-mobile to re-position your tooltip on small screens only.

Remember to replace {position} with: top, top-left, top-right, bottom, bottom-left, bottom-right, left, or right.

There are times when you need a 171px tooltip but constrained tooltips width is set to 240px by default.
You can add --tooltip-width: 171px as inline style and this will be applied to tooltips only.
Tooltips are built using pseudo elements and so are icons. In order to correctly use a tooltip in an icon it is necessary to wrap it.
By using the following markup to show tooltip on an icon, the result will be an empty/blank content among other issues. Although, for the purpose of this example the [icon] text was added where it should be an empty space since icons don't have content inside (between open and close tags).
[icon]
Wrap the icon on an empty div or span and assign it the tooltip class and data.

Documentation

Base Structure Tooltips are simple, you just need the base structure and add extra classes for special alignment. 
<!-- Base Structure -->
<div
	class="sui-tooltip"
	data-tooltip="Tooltip content goes here"
>
	Element content
</div>
Variations Sample code to position your tooltips based on design requirements.
In case you have some doubts on what you need, go to "Samples" tab and take a look at the different tooltip variations.

Constrained There are cases when you need tooltips with an specific width. 
<!-- Constrained: Default Width -->
<button
	class="sui-button sui-constrained"
	data-tooltip=""
>
	Default
</button>

<!-- Constrained: Custom Width -->
<button
	class="sui-button sui-constrained"
	style="--tooltip-width: 170px;"
	data-tooltip=""
>
	Default
</button>
Special Fixes You can further modify tooltips using the following inline styles. 
<!-- Tooltips Width -->
<button
	class="sui-button"
	style="--tooltip-width: 170px;"
	data-tooltip=""
>
	Default
</button>

<!-- Tooltips Width (Small Screens Only) -->
<button
	class="sui-button sui-button-mobile"
	style="--tooltip-width-mobile: 170px;"
	data-tooltip=""
>
	Default
</button>

<!-- Tooltips Text Align -->
<button
	class="sui-button"
	style="--tooltip-text-align: left;"
	data-tooltip=""
>
	Default
</button>
